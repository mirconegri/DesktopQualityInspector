# Assets

Drop here any media referenced by the README, typically:

- `demo.gif` — a short screen recording of `live_inference.py` correctly
  classifying `lever_open` / `lever_closed` in real time. This is the single
  highest-impact addition you can make to this repo for a recruiter or tech
  lead skimming GitHub: a 5-10 second looped GIF showing it actually working.

Tip: record with any free screen recorder (e.g. Windows' built-in Xbox Game
Bar, `Win+G`), trim to a few seconds, then convert to GIF with a free tool
like ezgif.com to keep the file small enough for a README to load fast.
